# 🐧 Linix-Commands-

> A comprehensive, well-organized repository covering **Linux commands, tools, and administration techniques**.

This repository contains detailed explanations, practical examples, best practices, and configuration references for both beginners and advanced Linux users.

---

## 📚 What's Inside

This repository is organized into multiple learning modules and guides:

| Folder / File | Description |
|---------------|-------------|
| **LinuxProcess/** | Complete Linux Process Management Course |
| **Docker/** | Docker Basics Course |
| **Kali-Linux-And-Security/** | Security, Forensics & Kali Linux guides |
| **AI-Tools/** | Local AI models and tools setup guides |
| **00-00 to 00-06** | Core Linux fundamentals (Kernel, Filesystem, Users, Files, Jobs, etc.) |

## 📖 How to Use This Repository

1. **Clone the repository**
   ```bash
   git clone https://github.com/efxtv/Linix-Commands-.git
   cd Linix-Commands-
   ```

2. **Start Learning**
   - Begin with the numbered class files (`00-00`, `00-01`, etc.)
   - Follow the structured courses in each folder
   - Practice commands directly in your terminal

3. **Contribute**
   - Found a mistake or want to add more content?
   - Open an issue or submit a pull request!
